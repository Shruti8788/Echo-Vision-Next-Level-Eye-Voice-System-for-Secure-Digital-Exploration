# 👁️  Echo Vision : Next-Level Eye-Voice System for Secure Digital Exploration 

## 📌 Project Overview

**Echo-Vision** is an AI-powered browser automation system that enables hands-free interaction with **Instagram** and **YouTube** using **eye tracking, hand gestures, and voice commands**. The project combines **Computer Vision**, **Speech Recognition**, and **Browser Automation** to provide an intuitive and accessible user experience without requiring a traditional mouse or keyboard.

---

## 🚀 Features

* 👁️ Eye tracking for cursor movement
* ✋ Hand gesture recognition for browser navigation
* 🎤 Voice commands for browser control
* 🌐 Browser automation using Selenium WebDriver
* 📱 Hands-free control of Instagram Reels and YouTube Shorts
* ⚡ Real-time webcam and microphone-based interaction
* 🔒 Local processing for improved privacy
